# ベクトルまとめ

## 1. ベクトルの表記方法

ベクトルは大文字や小文字の太字で表されることが一般的です。例えば、$\mathbf{v}$ や $\mathbf{a}$ などのように表記します。また、ベクトルは要素の順序を重視するため、$\mathbf{v} = [v_1, v_2, v_3]$ のように要素をカンマで区切って表記することもあります。

## 2. ベクトルの演算

ベクトルには以下のような基本的な演算があります。

### ベクトルの加算

二つのベクトルの対応する要素を足し合わせます。要素ごとの演算なので、ベクトルの次元は一致している必要があります。例えば、$\mathbf{a} = [a_1, a_2, a_3]$ と $\mathbf{b} = [b_1, b_2, b_3]$ の和は次のように計算されます：

$$
\mathbf{a} + \mathbf{b} = [a_1 + b_1, a_2 + b_2, a_3 + b_3]
$$

### ベクトルの減算

二つのベクトルの対応する要素を引きます。要素ごとの演算なので、ベクトルの次元は一致している必要があります。例えば、$\mathbf{a} = [a_1, a_2, a_3]$ と $\mathbf{b} = [b_1, b_2, b_3]$ の差は次のように計算されます：

$$
\mathbf{a} - \mathbf{b} = [a_1 - b_1, a_2 - b_2, a_3 - b_3]
$$

### スカラー倍

ベクトルの各要素をスカラー（実数や定数）で倍する演算です。例えば、$\mathbf{v} = [v_1, v_2, v_3]$ をスカラー $c$ で倍すると次のようになります：

$$
c \mathbf{v} = [c v_1, c v_2, c v_3]
$$

### 内積

二つのベクトルの対応する要素を掛け合わせて足し合わせる演算です。内積はスカラー値となります。例えば、$\mathbf{a} = [a_1, a_2, a_3]$ と $\mathbf{b} = [b_1, b_2, b_3]$ の内積は次のように計算されます：

$$
mathbf{a} \cdot \mathbf{b} = a_1 b_1 + a_2 b_2 + a_3 b_3
$$

### 外積

三次元ベクトル同士のみで定義される演算です。二つのベクトルの外積は、直交するベクトルを求めるのに使用されます。$\mathbf{a} = [a_1, a_2, a_3]$ と $\mathbf{b} = [b_1, b_2, b_3]$ の外積は次のように計算されます：

$$
\mathbf{a} \times \mathbf{b} = [a_2 b_3 - a_3 b_2, a_3 b_1 - a_1 b_3, a_1 b_2 - a_2 b_1]
$$

## 3. ベクトルの性質

ベクトルにはいくつかの重要な性質があります。

### 長さ（ノルム）

ベクトルの長さはベクトルの要素の値から計算されます。ベクトル $\mathbf{v} = [v_1, v_2, v_3]$ の長さ（ノルム）は次のように表されます：

$$
\|\mathbf{v}\| = \sqrt{v_1^2 + v_2^2 + v_3^2}
$$

### 単位ベクトル

長さが1であるベクトルを単位ベクトルと呼びます。ベクトル $\mathbf{v}$ の単位ベクトル $\mathbf{u}$ は次のように計算されます：

$$
\mathbf{u} = \frac{\mathbf{v}}{\|\mathbf{v}\|}
$$

### 直交ベクトル

二つのベクトルが互いに直交するとは、それらの内積が0であることを意味します。$\mathbf{a}$ と $\mathbf{b}$ が直交する場合、$\mathbf{a} \cdot \mathbf{b} = 0$ となります。
