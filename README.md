# 3DX

This is the 3D engine production project.

![img.png](docs/README/img.png)

## 移動方法
+ `W` : 前進
+ `S` : 後進
+ `D` : 右へ移動
+ `A` : 左へ移動
+ `Space` : 上へ移動 **(正の方向にy軸移動)**
+ `Shift` : 下へ移動 **(負の方向にy軸移動)**

## 視点移動

視点はマウスで動かせます。
また、マウススクロールでズームが可能です

+ `X` : 視点リセット
+ `O` : ポリゴンのアウトラインのON/OFF

## ブロック

+ `BackSpace` : カーソル上にあるブロックを削除できます
+ `Delete` : 存在するブロックを全て削除します。
+ `R` : ランダムにブロックを生成します。
+ `Tab` : 地面の隆起が変化します。

## システム

+ `Escape` : システムが終了します。
